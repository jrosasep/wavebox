# WaveBox v0.30.0

Primera release pública de **WaveBox**, un laboratorio visual para explorar propagación ondulatoria en 2D sobre medios heterogéneos.

## Incluye

- simulación en tiempo real de una onda escalar 2D;
- potenciales, obstáculos duros, refracción efectiva y amortiguamiento;
- 21 presets;
- exportación MP4 de un preset único o de una mezcla de hasta 5 tramos;
- monitor de rendimiento y memoria;
- interfaz pensada para leer, tocar, modificar y extender.

## Idea del proyecto

WaveBox busca acercar ecuaciones diferenciales parciales, física ondulatoria y cálculo numérico a una experiencia visual manipulable. El proyecto también funciona como experimento sobre el uso de IA en simulación física y computación científica: usarla para prototipar, pero sin delegarle el criterio físico.

## Aviso honesto

Esta release modela una **onda escalar 2D**. No resuelve todavía la ecuación de Schrödinger, las ecuaciones de Maxwell ni ondas gravitacionales, aunque el motor está pensado para poder crecer en esa dirección.
